## 2. Brief Summary
An interactive Excel dashboard that analyzes over **190,000 Zomato restaurant records** to uncover business insights on restaurant performance, customer ratings, pricing, cuisines, and location trends.



## 3. Project Overview
The food delivery industry generates large volumes of data that can be used to understand customer preferences and restaurant performance. This project demonstrates how Microsoft Excel can transform raw restaurant data into meaningful business insights through data cleaning, analysis, and interactive dashboard design.

The dashboard allows users to explore restaurant data using filters and visualizations, making it easier to identify trends and support business decisions.

---

## 4. Problem Statement
Restaurant businesses need accurate insights to improve customer satisfaction, optimize pricing, and understand market demand. However, raw datasets are difficult to interpret without proper analysis.

This project aims to solve this problem by creating an interactive Excel dashboard that enables users to analyze restaurant performance, customer ratings, cuisine popularity, pricing, and geographical distribution.

---

## 5. Dataset
**Dataset:** Zomato Restaurant Dataset

**Total Records:** Approximately **190,000**

### Dataset Features
- Restaurant Name
- City
- Area/Location
- Cuisine
- Restaurant Type
- Average Cost for Two
- Rating
- Number of Votes
- Online Order Availability
- Table Booking Availability

### Data Cleaning Performed
- Removed duplicate records
- Handled missing values
- Standardized location names
- Corrected inconsistent text values
- Created calculated columns using Excel formulas

## 10. How to Run This Project

1. Download or clone this repository.
2. Open **Zomato_Dashboard.xlsx** in Microsoft Excel (2019 or Microsoft 365 recommended).
3. Enable Editing if prompted.
4. Use the slicers to filter data interactively.
5. Explore charts and KPIs to gain business insights.
